# Использование адаптивных методов прогнозирования в экономических исследованиях

Дипломная работа, защищённая на **«отлично»**.

## 📋 Описание проекта

Проект посвящён сравнительному анализу адаптивных методов прогнозирования временных рядов в экономических исследованиях. В рамках работы реализовано **18 моделей** трёх основных классов: статистические, нейросетевые и декомпозиционные методы.

### 🔬 Объект исследования
Временные ряды данных **Росстата РФ** (3 набора данных: серии A, B, C), содержащие экономические показатели с различными характеристиками:
- **Серия A**: данные с выраженной сезонностью и трендом
- **Серия B**: данные с трендовой компонентой
- **Серия C**: данные со сложной структурой

---

## 🎯 Реализованные модели

### Статистические модели (8)
| Модель | Описание |
|--------|----------|
| **ARIMA** | Авторегрессионная интегрированная скользящая средняя |
| **SARIMA** | Сезонная ARIMA с учётом сезонных компонент |
| **ARMA** | Авторегрессионная скользящая средняя |
| **AR** | Авторегрессионная модель |
| **MA** | Модель скользящего среднего |
| **Holt-Winters Additive** | Аддитивная модель Холта-Винтерса с автоматической оптимизацией параметров |
| **Holt / Holt Damped** | Модели двойного экспоненциального сглаживания (с затуханием и без) |
| **Simple Exponential Smoothing (SES)** | Простое экспоненциальное сглаживание |

### Адаптивные модели (5)
| Модель | Описание |
|--------|----------|
| **Adaptive SES** | Адаптивное экспоненциальное сглаживание с динамической настройкой параметров |
| **Recursive Least Squares** | Рекурсивный метод наименьших квадратов с фактором забывания |
| **Theil-Sen** | Робастная регрессия Тейла-Сена с обработкой тренда |
| **Naive Forecast** | Наивный прогноз (baseline для сравнения) |

### Нейросетевые модели (4)
| Модель | Описание |
|--------|----------|
| **LSTM** | Долгая краткосрочная память (Long Short-Term Memory) |
| **GRU** | Управляемые рекуррентные блоки (Gated Recurrent Unit) |
| **Vanilla RNN** | Классическая рекуррентная нейронная сеть |

### Декомпозиционные модели (3)
| Модель | Описание |
|--------|----------|
| **Additive Component** | Аддитивная декомпозиция ряда (тренд + сезонность + остаток) |
| **Multiplicative Component** | Мультипликативная декомпозиция ряда |
| **Mixed Component** | Смешанная декомпозиция компонентов |

## 📊 Метрики качества

Оценка моделей проводилась по трём основным метрикам:

| Метрика | Описание |
|---------|----------|
| **MAE** (Mean Absolute Error) | Средняя абсолютная ошибка |
| **RMSE** (Root Mean Square Error) | Корень из среднеквадратичной ошибки |
| **sMAPE** (Symmetric Mean Absolute Percentage Error) | Симметричная средняя абсолютная процентная ошибка |

### Дополнительные статистические тесты
- **Тест Люнга-Бокса** — проверка на автокорреляцию остатков
- **Тест Дарбина-Уотсона** — обнаружение автокорреляции в остатках

## 📈 Основные результаты

### Лучшие модели по сериям данных

| Серия | Лучшая модель | MAE (test) | RMSE (test) | sMAPE (test) |
|-------|--------------|------------|-------------|--------------|
| **A** | Theil-Sen | 0.0898 | 0.1134 | 3.10% |
| **B** | GRU | 3.3446 | 4.0344 | 3.34% |
| **C** | LSTM | 0.5168 | 1.2115 | 0.51% |

### Ключевые выводы
1. **Нейросетевые модели** (LSTM, GRU) показали наилучшие результаты на сложных нелинейных данных
2. **Адаптивные методы** (Theil-Sen, Adaptive SES) эффективны на данных с изменяющейся структурой
3. **Классические статистические модели** (ARIMA, SARIMA) остаются конкурентоспособными на стационарных рядах
4. **Декомпозиционные подходы** полезны для интерпретации компонент временного ряда

## 🛠️ Детали реализации

### Предобработка данных
- Проверка стационарности (ADF-тест)
- Логарифмическое преобразование для нестабильных дисперсий
- Нормализация MinMaxScaler для нейросетевых моделей
- Разделение на train/validation/test (70%/15%/15%)

### Особенности моделей
- **Holt-Winters**: автоматическая оптимизация параметров сглаживания через differential evolution
- **LSTM/GRU/RNN**: использование EarlyStopping и ReduceLROnPlateau для предотвращения переобучения
- **Theil-Sen**: робастная регрессия с окном сглаживания
- **Adaptive SES**: динамическая настройка коэффициентов сглаживания
